# Playro brand starter

## Brand
- Product name: **Playro**
- Positioning: AI-powered Roblox game builder
- Tone: simple, fast, creator-first, non-corporate

## Safe logo direction
Use an original symbol. Avoid Roblox-style tilted squares, block-R marks, official Roblox red/black palettes, and anything resembling Rojo branding.

## GPT Image 2 prompt
Create a clean original logo for **Playro**, an AI-powered Roblox game builder desktop app.

Requirements:
- original brand mark, not based on Roblox, Rojo, Discord, OpenAI, or any known game platform logo
- simple geometric icon paired with the wordmark "Playro"
- friendly, modern, memorable, startup-quality
- dark-mode friendly
- colors: electric violet, cool cyan, soft white on transparent or very dark background
- icon idea: flowing triangular play shape fused with a spark/forge trail, rounded geometry, minimal, balanced, readable at 32px
- typography: custom-looking sans serif, soft corners, high legibility
- no mascots, no controllers, no cubes copied from Roblox, no hexagons that look like existing crypto brands
- flat vector style, premium SaaS/game-creation aesthetic

Output:
- one centered logo lockup
- one icon-only variant
- transparent background if possible
